# blog_node_ejs
The project is a application of NodeJs in the initial learning stage 

##Tech Stack
Express
MongoDB Atlas  
EJX


Iwritz is a blog page where user with following functionality

1.Write an article
2.Edit an article
3.Delete an article
4.Add comments to each of these articles
